About

A Design System is the single source of truth which groups all the elements that will allow the teams to design, realize and develop a product. It serves as a guide for developers and designers to ensure that the platform is consistent in its visual feel and overall design.

Atomic Design

4GDS (4Gives Design System) makes use of the Atomic Design principle of Brad Frost (2013), which states that interfaces are composed of smaller elements or atoms. These atoms are used together to create molecules, which are then used to create organisms.

On the style guides, all the colors, typography, grid used in 4Gives are shown in this design system. Components in the design system are all defined and are segregated by atoms, molecules, and organisms and have guides on how to use them properly.

4GDS is a design system that compiles all components used in the development of the 4Gives platform. These components can be reused to create new elements so long as they follow the guidelines dictated in this system. Mobile versions of the components are also available in this design system.

Style Guides

A style guide establishes standard style requirements to improve communication by ensuring consistency both within a document, and across multiple documents. Because practices vary, a style guide may set out standards to be used in areas such as punctuation, capitalization, citing sources, formatting of numbers and dates, table appearance and other areas. The style guide may require certain best practices in usage, language composition, visual composition, orthography, and typography.

How to choose spacing

Coordinate small and large values, along with structural components like cards, to create visual groupings of related things. This helps merchants understand the interface and find what they are looking for.

For screens with specialized layouts, adjust overall spacing based on the density of the content. For example, a simple login screen on desktop display has more room to breathe, so more space can be used.

Do

Use less space between small components, or components that share a close functional relationship.

Do

Use more space between large components, or between components which are less functionally related.

Grid

All atoms, molecules, and organisms should follow the 8-point grid in order to create a consistent user interface. Use multiples of 8 to define dimensions, padding, and margin of both block and inline elements.

How to follow the 8-point grid

First define the grid overlay on a frame using the layout grid, then select the 8px setting to display the grid over the frame. Toggle the overlay on and off using the command Ctrl + Shift + 4.

For screens with specialized layouts, adjust overall spacing based on the density of the content. For example, a simple login screen on desktop display has more room to breathe, so more space can be used.

Do

Ensure that all elements follow the 8-point grid to maintain consistency.

Do

Ensure that block elements are sized correctly, and also follow the 8-point grid.

Atoms & Molecules

The building blocks of the design system, following the Atomic Design principle. All the components in 4GDS are composed of smaller elements we call atoms. Atoms are basic elements that cannot be broken down futher without losing their meaning or relevance. Molecules are groups of multiple atoms held together to create unique elements, and become more tangible and operational than single atoms.

Alerts

Alerts interrupt users with urgent information, details, or actions. Use alerts judiciously and sparingly since they can be disruptive. These are time-sensitive messages that are dismissible and can appear either statically when a page loads or dynamically in response to an action.

Success Alert

Used to notify the user that their action was successful.
